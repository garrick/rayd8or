= rayd8or

* www.commandline.org

== DESCRIPTION:

A server that monitors and synthesizes interesting bits of data
and publishes out to subscribed clients who can take all sorts of
usefull actions.

== FEATURES/PROBLEMS:

* Parsing for CruiseControl using HPricot (brute force)
* nice generic Rayd8orSample module for holding sample data
